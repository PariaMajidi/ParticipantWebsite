# Participant Website 


## Setup Environment

- download [node.js](https://nodejs.org/en/) (LTS is good).
- type command in terminal `npm install --global yarn` 

## Setup sounds 

Add sounds with .wav extension in folder _sounds_.

## Getting started 

On terminal
```bash 
yarn 

# on first terminal 
yarn start

# on second terminal
yarn frontend
```

```
ngrok http -region=us -hostname=solaria.ngrok.io 3001

```
